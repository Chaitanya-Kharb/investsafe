# 📈 InvestSafe — AI Investment Risk Simulator

A fintech web app that helps young Indian investors understand
risk before investing real money.

## What it does
- Runs 1000 Monte Carlo simulations on real NSE stock data
- Calculates loss probability across 6 time horizons
- Compares multiple stocks side by side
- Generates plain-language AI risk explanation using Groq LLaMA

## Tech Stack
Python · Streamlit · NumPy · Pandas · yFinance · Groq AI · Matplotlib

## Live Demo
[Click here to open the app](https://chaitanya-kharb-investsafe.streamlit.app)

## How to run locally
```bash
pip install -r requirements.txt
streamlit run app.py
```
